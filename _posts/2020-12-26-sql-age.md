# SQL로 나이 계산하기

> 나이는 매년 변하는 수치, DB에 저장하기 보다는 그때 구하자

YYYY-MM-DD 형식으로 저장되어있다고 가정

---

## 1. 태어난 연도로 나이 계산

```sql
select id,
	2020 - DATE_FORMAT(birthdate, '%Y') + 1 as age
from site_user
```


## 2. 생일을 고려하여 만 나이 계산

```sql
select id,
	DATE_FORMAT(FROM_DAYS(TO_DAYS(NOW())-TO_DAYS('birthdate')), '%Y') + 0 as real_age
from site_user
```
